# Руда

## Обычные астероидные пояса

Они делятся на три типа в зависимости от размера и плотности скопления астероидов. В английской версии игры их названия теперь унифицированы, чтобы их было проще различать:

- **Asteroid Belt** — самый маленький тип (аналог «Пояса астероидов»);
- **Asteroid Group** — средний по размеру («Группа астероидов»);
- **Asteroid Cluster** — самый большой и богатый («Скопление астероидов»).

Рядом с названием в овервью всегда стоит цифра (например, `Asteroid Cluster 1`), которая обозначает уровень пояса и напрямую зависит от статуса безопасности системы.

## Редкие (синие) пояса

В системах с низким и нулевым статусом безопасности могут появляться редкие аномалии, которые в овервью подсвечиваются синим цветом:

- **Condensed Asteroid Belt/Group/Cluster** — сжатые пояса, содержат **Rich Ore** (богатую руду), которая дает на 10% больше минералов при переработке.
- **A0 Rare Asteroid Belt** — крайне редкие спавны в нуль-секах, содержащие особо ценные типы руд.

## Лунные пояса (Moon Mining)

Лунные пояса не появляются сами по себе, они создаются игроками при помощи структур (Refineries). В овервью они называются:

- **Ubiquitous Moon Asteroid Belt** — самый распространенный тип лунного пояса.
- **Moon Asteroid Belt** — общее название для локаций, созданных после подрыва лунного куска.

**Основные типы лунной руды (для фильтров):**

- _Ubiquitous (Повсеместная):_ Zeolites, Sylvite, Bitumens, Coesite.
- _Common (Обычная):_ Cobaltite, Euxenite, Titanite, Scheelite.
- _Uncommon (Необычная):_ Otavite, Sperrylite, Vanadinite, Chromite.

Для твоего скрипта удобнее всего будет ориентироваться на ключевые слова **Cluster** (как приоритет №1), **Group** (№2) и **Belt** (№3).
