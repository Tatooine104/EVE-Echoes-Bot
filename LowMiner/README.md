
Проект предназначен для автоматизации добычи руды в лоу секторах. 

# Описание

Бот предназначен для действий в игре, запущенный на ОС Windows через эмулятор MEmu. 

Работа с другими эмуляторами не предусматривалась и не тестировалась *(по крайней мере на данный момент)*.

# Возможности

Робот обладает следующими возможностями:
- **Кликать в указанную точку** - программа посылает клик левой кнопкой мыши на указанную точку.
	- **Рандомная задержка** - в методе предусмотрена задержка перед действием в диапазоне от 1 до 5 секунд, для имитации действий человека. 
	- **Рандомное смещение** - в методе предусмотрено смещение клика относительно указанной точки в переделах 10 пикселей в обе стороны по осям X и Y.

# Структура

## Методы

Методы, реализованные в программе:
- **SmartClick** - метод отправки клика левой кнопкой мыши на указанную точку. Метод работает в фоне и не мешает пользователю. Перед кликом метод делает задержку в пределах от 1 до 5 секунд. Точка клика рандомно смещается относительно указанных координат в переделах 10 пикселей по обоим осям. 
- **ClickAt** - метод отправки в указанное окно клика левой кнопки мыши. 
- **ConsolePrint** - Выводит в консоль сообщение, подсвечивая если нужно указанным цветом.

## Функции

- **GetRandomDelayMs** - получить рандомное число в пределах от 1 до 5 по умолчанию для задержек перед действиями, чтобы имитировать будто играет человек. 
- **GetWindow** - Находит дескриптор окна по его заголовку и выводит информацию о его размерах.
