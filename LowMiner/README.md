
Проект предназначен для автоматизации добычи руды в лоу секторах. 

# Описание

Бот предназначен для действий в игре, запущенный на ОС Windows через эмулятор MEmu. 

Работа с другими эмуляторами не предусматривалась и не тестировалась *(по крайней мере на данный момент)*.

# Возможности

Робот обладает следующими возможностями:
- **Кликать в указанную точку** - программа посылает клик левой кнопкой мыши на указанную точку.
	- **Рандомная задержка** - в методе предусмотрена задержка перед действием в диапазоне от 1 до 5 секунд, для имитации действий человека. 
	- **Рандомное смещение** - в методе предусмотрено смещение клика относительно указанной точки в переделах 10 пикселей в обе стороны по осям X и Y.
