Проект предназначен для автоматизации рутинных операций в EVE Echoes. 

# Описание

Бот предназначен для действий в игре, запущенный на ОС Windows через эмуляторы:
- BlueStacks 5

Работа с другими эмуляторами не предусматривалась и не тестировалась *(по крайней мере на данный момент)*.

# Возможности

- [ ] **Глаз** - контроль наличия посторонних в системе и при обнаружении отправляет предупреждение в чат альянса.
- [ ] **Майнер** - добыча руды в лоу секторах.
- [ ] **Нигилист** - добыча руды в нигилах.
- [ ] **Краб** - зачистка аномалек в лоу.
- [ ] **Краб КБТ КАР** - зачистка КБТ аномалек в лоу на КАРе.
- [ ] **Краб КБТ Дред** - зачистка КБТ аномалек в лоу на дредноуте.
