Проект предназначен для автоматизации рутинных операций в EVE Echoes. 

# Описание

Бот предназначен для действий в игре, запущенный на ОС Windows через эмуляторы:
- BlueStacks 5

Работа с другими эмуляторами не предусматривалась и не тестировалась *(по крайней мере на данный момент)*.

# Возможности

| **V** | **Название**      | **Сценарий**      | **Описание**                                                                                      |
|:-----:| ----------------- | ----------------- | ------------------------------------------------------------------------------------------------- |
|       | **Глаз**          | LocalWatcher      | контроль наличия посторонних в системе и при обнаружении отправляет предупреждение в чат альянса. |
|       | **Майнер**        | LowMiner          | добыча руды в лоу секторах.                                                                       |
|       | **Нигилист**      | NigilMiner        | добыча руды в нигилах.                                                                            |
|       | **Краб**          | PVECobra          | зачистка аномалек в лоу на Rattlesnake/Cobra.                                                     |
|       | **Краб КАР**      | PVECarrier        | зачистка аномалек в лоу на КАРе *(любой КАР)*.                                                    |
|       | **Краб Дред**     | PVEDreadnought    | зачистка аномалек в лоу на дредноуте.                                                             |
|       | **Краб КБТ КАР**  | PVECAPCarrier     | зачистка КБТ аномалек в лоу на КАРе.                                                              |
|       | **Краб КБТ Дред** | PVECAPDreadnought | зачистка КБТ аномалек в лоу на дредноуте.                                                         |

# История версий

## v.0.01.000

- Реализована архитектура асинхронной многопоточной работы с несколькими аккаунтами.
- Реализована очередь задач для каждого аккаунта.
- Реализован массовый отварп всех кораблей, которые находятся в одной системе, где обнаружена опасность.
- Реализован сценарий "LocalWatcher" - контроль безопасности в системе и оповещение в чат альянса.
