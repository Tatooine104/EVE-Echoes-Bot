# 🛸 EVE Echoes Automation Bot

Программный комплекс для автоматизации рутинных операций и многопоточного управления аккаунтами в мобильной MMORPG **EVE Echoes**.

---

## 💻 Системные требования и окружение

Бот разработан для запуска на операционных системах семейства **Windows** и взаимодействует с игрой через Android-эмуляторы.

- **Поддерживаемые эмуляторы:** `BlueStacks 5` (Оптимизировано под разрешение 1280x720).
- **Взаимодействие:** Прямое управление эмулятором через фоновые команды `ADB (Android Debug Bridge)` и анализ графического интерфейса с помощью `OpenCV`.
- *Примечание:* Работа с другими эмуляторами (LDPlayer, Nox) на текущий момент не тестировалась и официально не поддерживается.

## 🛠 Архитектурные особенности

- **Thread-Safe Очередь задач:** Каждое окно имеет собственный независимый стек задач (`TaskQueue`), управляемый потокобезопасными блокировками (`lock`).
- **Умная кросс-аккаунт паника:** При обнаружении врага одним окном (например, скаутом `localwatcher`), сигнал тревоги мгновенно передается всем активным ботам в этой же звездной системе, принудительно очищая их текущую рутину и отправляя на безопасную станцию.
- **State Persistence (DTO):** Состояние каждого аккаунта (текущая задача, статистика срабатываний триггеров, общее время работы, локация `InSpace`) автоматически сериализуется в JSON-файлы для бесшовного восстановления работы после перезапуска.

## 📈 История версий

### v.0.01.000 (Базовый каркас безопасности)
- **Многопоточное ядро:** Реализована асинхронная архитектура `RunLoopAsync` для параллельной работы с несколькими аккаунтами. *(Текущие тесты проводились в однооконном режиме)*.
- **Диспетчер очередей:** Внедрена динамическая система управления задачами `AccountTask`. Состав и порядок действий автоматически запрашиваются у фабрики сценариев `ScenarioFactory`.
- **Глобальный модуль защиты:** Реализован алгоритм сквозного контроля безопасности. При панике боты прерывают текущую логику, выполняют команду экстренного отварпа и докаются на станцию.
- **Релиз скрипта LocalWatcher:** Полноценно запущен сценарий «Глаз». Бот успешно распознает маркеры отношений стендингов, контролирует интерфейс чата и использует быстрый ввод для оповещения альянса.
