# EVE Echoes Bot

Боты для EVE Echoes

В репозитории будут собираться скрипты для ботов, которые могут автоматизировать рутину в EVE Echoes.

Все скрипты и функции разнесены по разным файлам, в зависимости от назначения:
- **[[libUtility.au3|libUtility]]** - различные функции вспомогательного характера;

## libUtility

Различные функции вспомогательного назначения:
- **\_CW($sText)** - функция вывода текста в консоль с принудительной перекодировкой в UTF8.

- **\_GetHandleByTitle($sTitle)** - получить Handle окна эмулятора по его заголовку. Требуется для того, чтобы точно взаимодействовать с нужным окном эмулятора.

- **\_Log($sText, $lDebug = True, $sFilePath = "log.txt")** - вывести текст ошибки, в зависимости от состояние отладки или в консоль или в лог.

- **\_SetInnerSize($hWnd, $iWidth, $iHeight)** - изменить размер изображения игры внутри окна эмулятора под указанный размер.

- **\_MEmu_SetSize($hWnd, $iWidth, $iHeight)** - изменить размер окна эмулятора MEmu под указанный размер.

- **\_HumanSleep($iMin = 100, $iMax = 800)** - функция рандомной задержки для имитации действий человека.

- **\_SaveToIni($sSection, $sKey, $vValue, $sFilePath = "data.ini")** - функция записи данных в ini файл.
